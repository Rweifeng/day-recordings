﻿﻿﻿﻿﻿﻿# day-recordings

一个面向日常工作的桌面记录工具：把你当天处理过的**文件 / 文本 / 链接**快速沉淀成时间轴，支持收藏、搜索筛选、回收站、导入导出与路径迁移。

## 1. 当前功能总览

- 按日期记录：左侧日历选择日期，右侧时间轴查看当天内容。
- 快速录入：拖入文件/文字/链接，或手动输入文本保存。
- 文件本地落盘：Electron 模式下，拖入文件会真实保存到软件目录。
- 链接直开浏览器：链接类型通过系统默认浏览器打开。
- 收藏与重复：可收藏记录；重复内容自动标记并可高亮同类项。
- 搜索与筛选：支持关键词、类型、收藏、重复、回收站筛选。
- 标签系统：录入时可加标签（逗号分隔），时间轴支持标签点击反查。
- 回收站机制：删除进入回收站，可恢复或彻底删除。
- 导入导出：支持 JSON 备份导出和导入（可设置导入跳过重复）。
- 路径重映射：迁移后文件路径失效时可批量重映射。
- 布局模式：普通 / 紧凑 / 超紧凑，一键切换并记忆。
- 置顶开关：窗口可切换“始终置顶”。
- 快速唤起：托盘菜单 + 全局快捷键 `Ctrl+Shift+D` 聚焦输入框。
- 可见状态反馈：保存状态提示 + 非阻塞 Toast 提示。
- 每日快照备份：自动生成日快照，主记录损坏时可回退恢复。

## 2. 界面说明

- 左上：日历（月切换、日期点选、记录日期标记点）。
- 左下：输入/拖入区（文件选择、标签输入、文本输入、操作按钮）。
- 右侧：时间轴（筛选、搜索、分页加载、记录卡片预览）。

## 3. 记录类型与行为

### 文本

- 直接展示内容。
- 支持标签。

### 链接

- 识别为 `http/https` 后按链接展示。
- 点击后使用系统默认浏览器打开。

### 文件

- 展示文件名、大小、类型、路径。
- 支持“打开文件”“复制路径”。
- 图片文件会显示缩略预览。
- 文件不存在时会标记“文件失效”。

## 4. 设置项

设置面板包含：

- 导入时跳过重复。
- 默认布局（普通 / 紧凑 / 超紧凑）。
- 路径重映射（旧根路径 -> 新根路径）。

## 5. 数据与目录

启动后会在项目目录生成 `day-recordings-data`：

- `day-recordings-data/records.json`：主记录数据。
- `day-recordings-data/files/`：拖入保存的文件。
- `day-recordings-data/backups/`：每日自动快照备份。
- `day-recordings-data/last-backup.json`：快照元信息。

> 迁移方式：直接拷贝整个项目目录即可连同数据一起迁移。

## 6. 快捷键与效率操作

- 全局快捷键：`Ctrl+Shift+D`（快速唤起并聚焦输入框）。
- 输入框：`Ctrl+Enter` 快速保存文本。
- 页面：按 `/` 快速聚焦搜索框。

## 7. 运行方式

### 7.1 桌面模式（推荐）

1. 安装依赖：

```bash
npm install
```

2. 启动：

```bash
npm start
```

### 7.2 组件模式（小窗常驻）

```bash
npm run start:widget
```

特点：小窗、可置顶、适合桌面常驻快速记录。

### 7.3 浏览器模式（兼容）

可直接打开 `index.html` 或本地静态服务预览；但文件落盘等桌面能力以 Electron 模式为准。

## 8. Windows 打包

### 8.1 安装依赖

```bash
npm install
```

### 8.2 生成安装包（推荐）

```bash
npm run dist:win
```

### 8.3 生成便携版

```bash
npm run dist:win:portable
```

### 8.4 同时生成安装包 + 便携版

```bash
npm run dist:win:all
```

### 8.5 时间戳目录打包（规避目录占用冲突）

```bash
npm run dist:win:ts
```

会生成类似 `release-YYYYMMDD-HHMMSS/` 的输出目录。

## 9. 常见问题

### 9.1 `app.asar` 被占用无法覆盖

- 先关闭运行中的 `day-recordings/electron` 进程。
- 优先使用 `npm run dist:win:ts` 打包到新目录。

### 9.2 链接拖入后被系统拦截

- 已处理 `.url` 解析，链接会按 URL 记录并使用默认浏览器打开。

### 9.3 迁移后文件打不开

- 在设置中使用“路径重映射”，将旧根路径批量替换为新根路径。

## 10. 技术说明（简要）

- 桌面框架：Electron
- 前端：原生 HTML / CSS / JavaScript
- 数据存储：JSON 文件（记录） + 本地文件目录（附件）
